Interactive Tools

A collection of custom web browser applications by Kupkovaaa.

Built as small experimental instruments for sound, typography, rhythm, and visual systems. Each tool opens as a focused browser experience.

01 Reactive Typography

SOUND—TYPE Reactive Visualizer by Kupkovaaa

Video Preview Sound Input

A browser-based audio visualizer that turns sound into kinetic typographic motion. The experience is tuned for expressive letterforms, quick visual feedback, and a clean stage for live audio input.

02 Reactive Pattern

SOUND—PATTERN Reactive Visualizer by Kupkovaaa

Video Preview Pattern Engine

A generative visual system for translating sound into structured graphic patterns. Built for rhythmic compositions, abstract texture studies, and fast browser-based experimentation.

TRY SOUND—PATTERN
03 Chladni Study

SOUND—PATTERN CHLADNI Visualizer by Kupkovaaa

Video Preview Frequency Form

A Chladni-inspired sound visualizer focused on resonance, frequency, and geometric interference. It gives the collection a more scientific, frequency-led visual language.
